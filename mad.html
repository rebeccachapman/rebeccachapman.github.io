I created a program in MATLAB that computes MAD and its three components: Quantity, Element, and Stratum. The program also outputs these components at multiple spatial resolutions.
This program requires that TIF or GeoTIFF files be used as the X, Y, and Stratum inputs. Outputs include an excel sheet, bar graph, and scatter plot.

Download MAD Version 1.0 here!

NOTE: Must have MATLAB installed to run program. Files must be in the same folder path. The zip file will also contain sample data for two time points, as well as the excel output showing the values of each component. I encourage you to look at the attached help.pdf, which will aid in conceptualizing MAD using the Sea Surface Temperature data provided.
