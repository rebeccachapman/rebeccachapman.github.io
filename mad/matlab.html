I created a program in MATLAB that computes MAD and its three components: Quantity, Element, and Stratum. The program also outputs these components at multiple spatial resolutions.
This program requires that TIF or GeoTIFF files be used as the X, Y, and Stratum inputs. Outputs include an excel sheet, bar graph, and scatter plot.
